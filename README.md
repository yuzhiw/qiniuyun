# 证书自动导入工具
## Version：
`python 2.7`

## Usage:
安装依赖（全局指令）:
```
pip install -r requirements.txt
```

运行程序（当前目录）：
```
python main.py
```

## 配置项 config.py
`config.py` 文件内定义了
1. 七牛云的连接参数
2. 证书模板的存放目录
3. 证书上的姓名和编号的位置，颜色，大小，字体等参数
4. 优秀学院和普通学员的名单位置
